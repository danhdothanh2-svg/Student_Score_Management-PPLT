# 🎓 Student Score Management System

## 📌 Description
This is a simple Python program to manage student scores.  
It allows users to add, search, sort, and analyze student data.

---

## 🚀 Features
- Add student (ID, name, score)
- Display student list
- Search by ID or name
- Sort students by score
- Basic statistics (average, max, min)
- Advanced search
- Grade classification (A, B, C, D)
- Save/load from text file
- Save/load from JSON file

---

## 🛠 Requirements
- Python 3.x

---

## ▶️ How to Run
1. Open terminal in project folder  
2. Run the program:
python your_file_name.py
